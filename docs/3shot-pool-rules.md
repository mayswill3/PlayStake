# 3-Shot Pool -- Rules

## Overview

Each player gets **3 shots**. Every ball you pot scores 1 point. Highest score wins.

---

## Setup

- Standard 15-ball rack on a 6-pocket table.
- Player 1 (P1) always breaks first.

## How to Play

1. **Player 1** takes 3 consecutive shots. Shot 1 is the break.
2. When P1 finishes, the board stays as-is -- balls P1 potted are gone, remaining balls stay where they are.
3. **Player 2** takes 3 consecutive shots on the remaining balls. The cue ball is placed at the head position for P2's first shot.
4. The player with the higher score wins.

## Scoring

| Event | Points |
|-------|--------|
| Ball potted | +1 |
| 8-ball potted | +1 (no special value) |
| Cue ball pocketed (scratch) | 0 -- but the shot is consumed |
| Balls potted on a scratch | Still count |

**All 15 balls are worth the same.** No solids, no stripes, no groups. Just pot balls.

## Scratches

- A scratch (cue ball pocketed) **uses up your shot**.
- Any object balls potted on the same shot **still count** toward your score.
- The cue ball is respotted at the head position for your next shot.

## Shot Clock

- You have **30 seconds** per shot.
- At 3 seconds the timer turns red and pulses.
- At 0 seconds an auto-shot fires at minimum power in whatever direction you were aiming.

## Tiebreaker

If scores are tied after regulation:

### Overtime -- Sudden Death
- Each player takes **1 shot**. Player who pots more wins.
- If still tied, another round. Up to **3 overtime rounds**.

### Forced Tiebreak (after 3 OT rounds)
- A target appears at the center of the table.
- Each player shoots the cue ball -- **closest to center wins**.
- If the cue ball is pocketed, distance counts as maximum (you lose).
- If distances are exactly equal, P2 wins (to offset P1's break advantage).

A single winner is **always** produced. No draws.

## Edge Cases

| Situation | What happens |
|-----------|-------------|
| All 15 balls potted before shots used up | Remaining shots are forfeited. If P1 cleared the table, P2's turn is skipped (score: 0). |
| Cue ball and object ball pocketed same frame | Object ball counts (+1), scratch consumes the shot, cue respots. |
| No balls contacted (whiff) | Shot is consumed. No penalty beyond the wasted shot. |
| Overtime with no object balls left | Skips straight to forced tiebreak (closest-to-center). |
| Cue ball respot overlaps an object ball | Cue ball shifts along the head string until a clear spot is found. |

## Player 2 Asymmetry

P2 knows P1's score before shooting. This is intentional -- it creates tension ("Can I beat 3?") and strategic adaptation (play risky if behind, safe if ahead). P1's advantage is a full rack with clustered balls; P2's advantage is information.

## Quick Reference

- **Shots per player:** 3
- **Points per ball:** 1
- **Shot clock:** 30 seconds
- **Overtime:** Up to 3 rounds of sudden death, then closest-to-center
- **Target game time:** 60--90 seconds
